# BraveStory

一个使用 Miros 插件复刻的 RPG 游戏 DEMO，主要作用是用来协助开发与测试该插件 .

教材来源于：[timothyqiu|《勇者传说》Godot 4教程合集](https://space.bilibili.com/7092/channel/collectiondetail?sid=1304862)

Godot Edition：v4.3-stable_mono_win64

>  不能使用 v4.3 及以上版本进行开发，因为当前场景仍然使用 Tilemap 与 ParallaxLayer ，两者在 v4.3 中已经被新节点所取代了，所以更新到 v4.3 以上版本存在显示上的 BUG .
